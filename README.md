# Qubox - Toy Quantum Circuit Editor

A graphical quantum circuit editor built with PyQt6 that allows users to design and simulate quantum circuits.

## Features

- Visual quantum circuit design interface
- Bloch sphere visualization for quantum states
- Import/export circuits in QASM format
- Simulation of quantum operations
- Customizable quantum gates

## Installation

1. Clone the repository:
```
git clone https://github.com/ronaldossai/Qubox-Toy-Quantum-circuit-editor.git
cd Qubox-Toy-Quantum-circuit-editor
```

2. Install required packages:
```
pip install -r requirements.txt
```

3. Run the application:
```
python src/main.py
```

## Dependencies

- Python 3.x
- PyQt6
- Matplotlib
- (other dependencies listed in requirements.txt)
